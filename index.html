Get started

Install in
60 seconds.

Free forever. No account needed. Just a free API key from Groq.

1

Clone the repo

Download from GitHub or clone with git clone

2

Load in Chrome

Go to chrome://extensions → Enable Developer Mode → Load Unpacked → select the folder

3

Get your free API key

Sign up at console.groq.com → API Keys → Create API Key. It's completely free.

4

Enter key in Clarify

Click the Clarify icon in your toolbar, paste your key, click Save.

5

Select any text

Go to any webpage, select a word or sentence, and watch the magic happen.
